SCOPE & AUTHORISATION

RedShadow V4 performs active and passive reconnaissance including:

  - Subdomain enumeration and DNS bruteforce
  - Port scanning via Nmap
  - HTTP probing and vulnerability detection
  - Secret and credential scanning
  - S3, GCS, and Azure bucket discovery
  - JavaScript endpoint extraction
  - Open redirect detection
  - Subdomain takeover detection
  - Wayback Machine URL analysis
  - GitHub secret scanning
  - CVE analysis with NVD and EPSS enrichment

Every capability in this tool has the potential to generate network
traffic directed at the target. You must hold explicit written
authorisation from the target system owner before running any scan.
